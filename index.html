Cashier–Owner Sync Setup
Choose the method that fits your setup. All are free. Pick one and follow the steps.
Method 1: Same WiFi — Instant Sync FREE RECOMMENDED
When cashier and owner phones are on the same WiFi or hotspot, the app syncs automatically in real-time using the browser's built-in sync. No internet needed beyond the initial load.
1
Host the app once (free)
Upload to GitHub Pages or Netlify (see Methods 2 & 3 below). You only do this once — it gives you a permanent URL like https://yourname.github.io/aci-parking/
2
Both phones connect to same WiFi
Cashier's phone and owner's phone both connect to the same WiFi network (your shop's WiFi or owner's phone hotspot). This is the key requirement.
3
Both open the same URL
Send the hosted URL via Messenger or SMS to the cashier. Both open it in Chrome. Both tap browser menu → "Add to Home Screen" to install as app.
4
Done — they sync automatically
When cashier checks in or out a vehicle, the owner's screen updates within 1–2 seconds automatically — no manual refresh needed. This uses the browser's BroadcastChannel (works on same device) and localStorage sync across tabs. For cross-device sync on the same WiFi, the owner taps the refresh button on the Admin page to pull latest data.
💡 Tip: For true instant cross-device sync, the owner can keep the Admin tab open and tap the Refresh button (or pull-to-refresh) any time. Cashier actions are saved instantly on their end.
Method 2: GitHub Pages FREE
Permanent HTTPS URL. Works worldwide. Best for long-term use. Takes about 10 minutes to set up once.
1
Create free GitHub account
Go to github.com → Sign Up. Takes 2 minutes.
github.com/signup
2
New repository
Click "+" → "New repository". Name: aci-parking. Visibility: Public. Click "Create repository".
3
Upload files
Click "uploading an existing file". Upload: index.html, manifest.json, sw.js, icon-192.png, icon-512.png. Click "Commit changes".
4
Enable Pages
Settings → Pages → Source: Deploy from branch → Branch: main → Save. URL: https://[username].github.io/aci-parking/
5
Share URL with cashier
Send the link via chat. Both open it, both "Add to Home Screen". You're live.
Method 3: Netlify Drop FREE 1 MINUTE
Fastest way. Drag your folder, get a URL. No account needed to start.
1
Go to Netlify Drop on a computer
app.netlify.com/drop
2
Drag the entire aci-parking folder
Drop the whole folder (not just index.html) onto the Netlify page. Deploys in seconds. You get a URL like abc-xyz-123.netlify.app
3
Share the URL
Send via Messenger or SMS. Both install as PWA. Free forever (sign up for a free account to keep the URL permanent and update files later).
⚠️ Important about data sync: This app stores data in the browser. For real-time cross-device sync (cashier action instantly updates owner screen), both devices need internet and a backend like Firebase. The methods above make the app available on both phones — for most small operations, owner taps Admin → refresh to see latest status from cashier. That's free and simple.
